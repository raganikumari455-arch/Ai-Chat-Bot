# 🤖 Rule-Based AI Python ChatBot

A simple rule-based chatbot built using Python. This project is designed for beginners to understand how chatbots work using basic programming concepts like dictionaries, loops, and functions.

---

## 📌 Features

* Greets the user based on the current time (Morning, Afternoon, Evening, Night)
* Takes user input interactively
* Responds using predefined rules (dictionary-based responses)
* Simple and beginner-friendly logic
* Exit option using `"bye"`

---

## 🛠️ Technologies Used

* Python 🐍
* Standard Libraries:

  * `datetime`
  * `time`

---

## 🚀 How It Works

1. The chatbot asks for your name.
2. It greets you based on the current system time.
3. You can ask basic questions.
4. The bot checks your input against predefined responses.
5. If it finds a match, it responds accordingly.
6. If no match is found, it gives a default response.
7. Type `bye` to exit the chatbot.
